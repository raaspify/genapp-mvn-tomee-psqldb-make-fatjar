This page comes from ThemesBrowse to edit an Item. Each theme or a shopping item is defined and described in Item table with a link to the actual image in blobdata table or text with image in clobdata table. So this page has edits related to item table only and covers themes as products and its summary, specification and reviews information. It does not cover theme contents which are in clob or blob data tables. This page is shown to backoffice Users and to visitors if social sharing site with high to medium level roles. Theme vendors ie PH type and role creates and modifies themes and its content but do not have access to edit product info for a theme.
